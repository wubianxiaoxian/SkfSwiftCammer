###在oc中调用swift文件方法
   1. 在oc中使用swift文件
   2. 创建全局编译文件 PrefixHeader.pch
   3. 在全局预编译文件PrefixHeader.pch中 #import ‘项目名-Swift.h")
   4. 现在就可以在oc中使用swift了
